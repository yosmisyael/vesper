# vesper: Modern and Elegant Hyprland Ricing for Unparalelled Productivity

## Features
- Application launcher
- Theme dynamically adapt to wallpaper
- System power control
- Hardware performance manager
- Clipboard manager
- Screenshoot utility (along with editing feature with [swappy](https://github.com/jtheoof/swappy))
- Screen sharing utility
- Notification center (credit to [HyprNova](https://github.com/zDyant/HyprNova))
- Wallpaper selector


## Installation
clone this repo, then run the install scripts inside install dir (no backup will created, fresh arch hyprland recommended)
